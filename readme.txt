A complete restaurant ordering & management system — menus, delivery, reservations, POS, and AI analytics, all in WordPress.

== Description ==

Dish Dash is an all-in-one restaurant operating system built on top of WordPress and WooCommerce.

**Core Features (Phase 1):**
* Visual menu display with category filter and live search
* Custom Post Type for menu items with price, badges, allergens, and nutrition
* Add to cart functionality
* Full admin dashboard with setup checklist
* Settings panel for currency, tax, and feature toggles

**Coming in Future Phases:**
* Online ordering with delivery & pickup
* Table reservations with QR codes
* Multi-branch management
* POS terminal
* AI-powered analytics (Claude AI)

== Installation ==

1. Upload the `dish-dash` folder to `/wp-content/plugins/`
2. Activate the plugin through the **Plugins** menu in WordPress
3. Go to **Dish Dash → Settings** to configure your restaurant
4. Add menu items via **Dish Dash → Menu Items**
5. Add the shortcode `[dish_dash_menu]` to any page

== Frequently Asked Questions ==

= Does this require WooCommerce? =
Yes. WooCommerce is required for cart and payment processing.

= Can I override the templates? =
Yes! Copy any template from `/plugins/dish-dash/templates/` to `/your-theme/dish-dash/` and edit freely.
